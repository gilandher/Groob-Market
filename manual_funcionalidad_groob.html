📘 Manual de Funcionalidad

Guía Operativa del E-Commerce — Groob Market

Groob Market es una plataforma de e-commerce completa, diseñada para maximizar la conversión en ventas a través de una interfaz de usuario atractiva y un panel administrativo robusto. A continuación, se explican sus funciones principales.

1. Funcionalidad para el Cliente (Experiencia de Usuario)

🔍 Exploración y Compra

El cliente navega por un catálogo categorizado. Puede buscar productos utilizando una barra predictiva que ignora tildes (ej: "Audifonos" = "Audífonos"). Cada producto cuenta con insignias dinámicas, precios tachados si hay descuento e indicador de "Bajo Stock".

🎁 Fidelización: Wishlist y Ruleta

Favoritos (Wishlist): Los clientes pueden marcar productos con el ícono de ❤️. Esta lista persiste y es accesible en cualquier momento desde el menú superior.

Ruleta de Premios: Al ingresar, el usuario ve una ruleta "Gira y Gana". Solo se permite un tiro por sesión/usuario. Si gana un descuento (ej: 10%), el código se genera automáticamente. Si el usuario no está registrado, el código se enmascara parcialmente (GROOB-****) invitándolo a crear una cuenta para reclamarlo.

🚚 Checkout y Cotizador de Envíos en Tiempo Real

En el carrito de compras, el cliente ingresa su municipio. El sistema consulta instantáneamente la base de datos de Zonas de Cobertura para confirmar si llega y cuánto cuesta. Si la ciudad ingresada no está registrada, el sistema sugiere automáticamente "Cotizar por WhatsApp", abriendo el chat oficial con un mensaje pre-armado con todos los detalles del pedido.

📄 Gestión de Pedidos y Descarga PDF

Los usuarios registrados acceden al portal /orders para seguir su producto en tiempo real mediante una Barra de Progreso Logístico (Nuevo -> Confirmado -> Empacando -> En Camino -> Entregado).

Tienen la capacidad de descargar en formato PDF:
• Pre-factura: Para cuando el pedido apenas fue realizado.
• Factura Comercial: Para cuando el pedido se encuentra completado, la cual viene acompañada del marco legal de garantía para compras online (Ley 1480 de 2011).

2. Funcionalidad para el Administrador (Operador del Negocio)

🎮 Panel de Control Automático (Jazzmin Admin)

El administrador accede al backend (/admin) desde un portal altamente visual y fácil de usar, sin necesidad de saber programar. Desde allí puede administrar todo el inventario de la tienda.

📦 Gestión de Tarifas de Envío sin Código

El administrador puede ir a la sección "Tarifas de Envío" y administrar hacia dónde se envía. Puede agrupar por ejemplo "Envigado" y "Sabaneta" en la ZONA 3 y asignarles un precio estricto de envío ($8.500 COP). El calculador de la web en el frontend asimilará estos cambios de forma instantánea al guardarse desde este administrador, logrando total independencia técnica.

📋 Procesamiento Seguro de Pedidos

El área administrativa almacena la información vital (cuántos productos se vendieron, costo de envío, método de pago, información del usuario, etc). Una vez en el sistema, el administrador simplemente cambia el estado a (Confirmado, Despachado, etc.) y toda la plataforma (frontend) se actualiza de manera automática.
